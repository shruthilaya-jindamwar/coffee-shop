# Coffee Shop ☕

A simple coffee shop website demo.

## Features

1. **Menu Page**
   - Users can view different coffee items.

2. **Add to Cart**
   - Users can select items from the menu and add them to the cart.

3. **Cart Page**
   - Users can see all selected items in their cart.

4. **Support the Project**
   - On the cart page, users will find a **Buy Me a Coffee ☕** button to support the developer.

## Purpose

This project demonstrates a basic **shopping cart flow**:

Menu → Add to Cart → Cart Page → Support the Developer

It is a demo project and does not process real purchases.
